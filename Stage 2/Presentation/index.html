Angular

What is Angular?

Client side Javascript framework, create and maintained by Google, used to build powerful single page applications(SPA)

Features

  • React speed of working
  • Mobile first
  • Easy to learn
  • Easy to maintain
  • Typescript

Main construction blocks

  • Modules
  • Components
  • Templates
  • Metadata
  • Data Binding
  • Directives
  • Services
  • Dependency Injection
